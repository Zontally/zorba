# Changelog

All notable changes to ZORBA will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/), and this project adheres to [Semantic Versioning](https://semver.org/).

## [0.1.0] — 2026-03-18

### Added

- **Framework documents:** Manifesto, Architecture (v0.2), Blended Workforce Model, Domain Reference Model, Agentic Enterprise Patterns, Governance & Trust Framework, Information Model, Industry Editions, Glossary
- **Domain taxonomy:** 12 detailed domain files with ~550 processes, each with unique 6-digit ID and agentic profile
- **Architecture model:** OU → Domain → Capability → Process → Activity → Work with Strategy as cross-cutting alignment overlay
- **Organisational Unit** as root object type — recursive, supports complex corporate structures
- **Value Chain / Management Function** domain classification with value-driven subtitles
- **Framework-agnostic information model** supporting ZORBA, APQC/PCF, or custom taxonomies
- **Industry Editions** concept with "ZORBA for Technology Service Providers" as flagship
- **ZORBA-specific capabilities:** Blended Workforce Management, Agent Platform Management, AI Governance & Agent Compliance
- **Website components:** Homepage sections as Tailwind CSS HTML components
- **GitHub Actions:** Semantic versioning and release workflow
